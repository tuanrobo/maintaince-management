Overview

Để đáp ứng các bộ phận kỹ thuật bán hàng tự động và giải quyết nhu cầu như: sửa chữa, nâng cấp, bảo trì về các thiết bị phần cứng của máy bán hàng tự động. Chúng tôi phân tích và cung cấp các giải pháp để tạo ra các công cụ hỗ trợ bộ phận kỹ thuật để giải quyết các vấn đề trên.

Expectation (Requirement draft from @Tin)

Spare parts management

Input parts list with quantity.

Each time Technician completes replace a part, he will confirm on Maintenance function on the machine.

And the machine push this information to server.

The server auto decreases quantity of this part on server and records it.

At the same time, ID machine/time done /ID of Technician to server.

Launch Demo

Quản lý linh kiện/vật tư cho máy bán hàng

Hệ thống quản lý linh kiện/phụ tùng cho máy (Nhập mới/Thay thế)

Nhân viên kỹ thuật xác nhận sau mỗi quá trình sửa chữa/thay thế trên máy

Máy bán hàng sẽ chuyển thông tin sửa chữa này về server

Đồng thời, hệ thống ghi nhận lại các thông tin: staffID, machineID...

Monitoring dashboard

  1. Note jam
  2. Temperature inside the cabinet
  3. Door Status
  4. Slot broke
  5. Network status
  6. Maintenance Status(What machine need to clean Bill Validator clean Cooling system on next month...),
  7. System status

Notification alert needs to push for Technician group:

  1. Note Jam
  2. Door Status
  3. Lost connection between machine and server more than xx hours
  4. Temperature inside the cabinet above xxx

Bảng điều khiển giám sát

  1. Kẹt tiền
  2. Nhiệt độ
  3. Hiện trạng cửa (Mở/ Đóng)
  4. Hỏng khay
  5. Trạng thái mạng (có /rớt/ chập chờn)
  6. Trạng thái bảo trì: Máy cần có quá trình bảo dưỡng theo tháng/quí/năm... tuỳ thuộc vào từng loại linh kiện thì sẽ có những mốc thời gian trùng tu bảo dưỡng khác nhau và được hệ thống cập nhật tự động)
  7. Trạng thái hệ thống

Hệ thống thông báo thời gian thực(real-time) đến bộ phận kĩ thuật về các vấn đề:

  1. Kẹt tiền
  2. Hiện trạng cửa máy bán hàng
  3. Mất kết nối từ máy bán hàng đến server trong một khoản thời gian nào đó
  4. Nhiệt độ máy có vấn đề

Statistics

Record database about:

  1. Note jam
  2. Slot broke, any problem affect to sale.

This information statistic help us predict what are spare parts more broken. Record date-time after maintenance machine (After Clean Bill Validator or Clean Cooling system).

Thống kê

Ghi nhận các vấn đề:

  1. Kẹt tiền
  2. Hỏng khay
  3. Các vấn đề ảnh hưởng đến quá trình vận hành máy

Hệ thống thông kê giúp cho việc dự đoán các loại linh kiện/vật tư có khả năng hư hỏng. Ghi nhận lại thời gian bảo trì máy.

Machine settings remotes

Some parameters we can set on the server-side and sync to the machine without going to the machine and set on the machine. Example: Temperature limit, the domination of the Bill validator can accept (we already have this function on our server), the time turn on/turn off the light, and LCD screen.

Cấu hình máy bán hàng từ xa

Một hệ thống để nhân viên có thể cấu hình máy từ xa mà ko cần đến nới đặt máy thủ công nhằm tiết kiệm thời gian và công sức. Ví dụ: Cần điểu chỉnh lại nhiệt độ máy, hiệu chỉnh lại mệnh giá (Đã có sẵn trên hệ thống), thời gian tắt/mở đèn và màn hình LCD
